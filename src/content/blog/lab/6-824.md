使用 `GoLand` 在 `wsl2` 上开发。

## Go 安装

我的 WSL 之前已经装过 Go：

```bash
go version

go version go1.19.4 linux/amd64
```

## 代码获取

```bash
git clone git://g.csail.mit.edu/6.824-golabs-2022 6.824
cd 6.824
ls
```

## 利用 git 进行版本管理

从 MIT 的仓库下载代码后，在自己的 GitHub 中新建一个仓库，然后把本地代码推到自己的仓库里：

```bash
git remote remove origin
git remote add origin <仓库 git>
git branch -M main
git push -u origin main
```

开分支留存源代码：

```bash
git branch archive
git push -u origin archive
```
